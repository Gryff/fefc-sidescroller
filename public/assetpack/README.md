# Pixel art character pack

Introducing my brand new Pixel Art  Character Pack - a pack of charming and spicy pixel art characters that will bring a touch and flair to your projects.  Best of all, it's completely free! Download now and unleash your creativity with these Pixel Art Characters and accessory layers!

There are 7 animations available for both Male and Female characters, along with layers for clothing, hair, and accessories. This setup provides plenty of options for creating unique character variations.

Includes the following animations in the Sprite Sheet: 80x64
- Idle (5 frames)
- Walk (8 frames)
- Run (8 frames)
- Jump (4 frames)
- Fall (4 frames)
- Use item/Attack (6 frames)
- Death (10 frames)

Character Layers: 
- Hand Items
- Arm Layers
- Elven ears
- Hair Layers
- Clothing
- Hat Layers
- Back layers
- Masks
- Special skins and more!

Extras: 
- Emojis,
